# piems
This is a command-line tools to operate on time intervals.

## installation
To install piems, use:
```
pip install git+https://github.com/kazet/piems
```

## examples
```
$ piems
🐕 1h 3m + 2h 5m + 10m
3h 18m
🐕 19:01 to 19:33
32m
🐕 19:01 to 19:33 + 1h
1h 32m
🐕 (1h 1m + 2h)
3h 1m
🐕 (1h 1m + 2h) * 0.5
1h 30.5m
```
